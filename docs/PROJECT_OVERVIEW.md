# Project Overview

## What This Site Is

A standalone proposal website presenting MVNE (Pty) Ltd's comprehensive MVNO enablement proposal for Centurion and Co. The site translates the source PPTX workshop presentation and DOCX proposal document into a structured, interactive web experience following the pattern of the BP MVNO PWA & USSD proposal site.

## Target Audience

- Centurion and Co decision-makers and stakeholders
- Internal MVNE / DSG Group review teams

## Deployment Environment

- **Platform:** Vercel
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4 + shadcn UI
- **Language:** TypeScript

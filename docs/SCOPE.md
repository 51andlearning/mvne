# Scope

## Pages Included

- `/` — Single-page proposal (all sections on one scrollable page)

| # | Section | Source |
|---|---------|--------|
| 1 | Hero / Proposal Overview | Cover page metadata |

## What Is Excluded

- Multi-page routing (all content on single page)
- Live e-signature integration
- Backend API / form submissions
- Partner portal or login functionality
- Wholesale rate appendices (commercially sensitive)
- Embedded PPTX slide images (not available as extracted assets)
